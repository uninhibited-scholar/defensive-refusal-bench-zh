# Prior-art（2026-06-20 联网复核）

**英文**该问题刚被确立为前沿，**中文安全垂类仍空白**（本基准定位）。

- **Defensive Refusal Bias: How Safety Alignment Fails Cyber Defenders**（arXiv:2603.01246，ICLR 2026 workshop）——用美国 NCCDC **英文**数据测网安场景的 False Refusal Rate。证明问题真实且重要，但英文。
- **OR-Bench**（arXiv:2508.11222 周边）/ **XSTest** / **FORTRESS** ——英文 over-refusal 基准，非安全垂类、非中文。
- **CSSBench**（arXiv:2601.00588，2026.01）——中文通用 over-refusal 子集，**未覆盖网安防御垂类**。
- 结论：**中文 · 网安防御 · 误拒 + 对照危害组 · 可机器评分** 的交叉点未见同类。
